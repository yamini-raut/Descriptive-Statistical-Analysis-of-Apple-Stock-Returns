# Descriptive Analysis of Apple Stock Returns

## Objective
Performed descriptive statistical analysis on Apple stock returns to evaluate central tendency, volatility, and distribution patterns.

## Dataset
Historical Apple price data (daily frequency).

## Methodology
- Converted stock prices into daily returns
- Performed descriptive statistical analysis
- -Visualized avg daily return
  
## Key Results
Graphical representation of avg daily return and closing price.


## Tools Used
- Python
- Pandas
- NumPy
- SciPy
